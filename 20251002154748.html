I. 월간 관심종목

몽고 DB(MDB): AI 시대에 적합한 데이터베이스 플랫폼

기업들의 LLM 사용 증가하면서 몽고 DB 데이터베이스 활용 가속화 몽고 DB는 문서형 데이터베이스 플랫폼으로 기존 데이터베이스 대비 성능, 확장성 측면에서 우위. 아울러, Vector 검색 + 임베딩 + Reranker 기능을 통해 LLM 검색에 최적화. 올해 초부터 기업들이 자사의 데이터에 기반한 LLM 사용이 늘어나면서 성장 가속화될 전망

문서형 데이터베이스 플랫폼으로 LLM 검색에 최적화 사업부문은 1) 구독 (97%), 2) 전문 서비스 (3%) 로 구분 AI 시대에 적합한 문서형 데이터베이스 플랫폼. 몽고 DB 데이터베이스는 문서형 아키텍처 위에 구축되어 기존 관계형 데이터베이스의 강점에 더해 성능, 확장성, 유연성을 보유. 앱을 빠르고, 비용 효율적으로 구축할 수 있도록 지원. 실시간 운영 데이터를 함께 처리하도록 설계

몽고 DB는 JSON 포맷 기반의 문서형 데이터베이스를 제공. JSON 포맷은 데이터를 구조화, 저장, 관리하는 방법을 정의한 설계 = 스키마 (Schema) 가 유연하다는 강점을 가짐. 이에, 비정형 데이터 처리에 유리. 몽고 DB는 JSON 포맷 기반 문서형 데이터베이스 위에 Vector 검색 + 임베딩 + Reranker 기능 연결해 LLM 검색에 최적화. AI 수요가 증가하면서 몽고 DB 수요 증가할 전망

고객들의 몽고 DB 서비스 사용량이 늘어나면서 10만 달러 이상을 지출하는 고객 수 꾸준히 증가. 고객 수 증가하는 가운데, 개별 고객의 사용량 늘어나면서 매출 성장률 반등하는 국면

가트너에 따르면 데이터 관리 소프트웨어 시장 규모는 2024년 930억 달러에서 2028년 1,500억 달러로 커질 전망. 몽고 DB 경영진은 AI 수요가 증가하고 있는 점을 고려해 시장 전망이 상향 조정될 것으로 예상. 몽고 DB 시장 점유율은 2% 수준으로 향후 점유율 상승 여력 충분
