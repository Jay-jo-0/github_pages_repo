마이크로소프트(MSFT): 하반기를 기약하는 마이크로소프트

FY 3Q26 Review: 코파일럿 수익화에 적극적으로 나서는 마이크로소프트

FY 3Q26 매출액은 829억 달러(+18.3% YoY, +2.0% QoQ)로 컨센서스를 1.8% 상회했습니다. 영업이익은 384억 달러(+20.0% YoY, +0.3% QoQ)로 컨센서스를 4.2% 상회했으며, EPS (GAAP 기준)는 4.27달러(+23.4% YoY, -17.2% QoQ)로 5.4% 상회했습니다.

Azure 클라우드와 생산성 및 비즈니스 프로세스 부문의 매출 성장률은 하반기에 모멘텀을 받을 전망입니다. 마이크로소프트는 지난 분기 실적 발표에서 Azure 클라우드 성장률이 기대에 미치지 못한 것은 사업 부문의 균형 잡힌 발전을 추구한 결과라고 설명했습니다. 마이크로소프트는 확보한 캐파를 1) M365 코파일럿, GitHub 코파일럿과 같은 자사 앱, 2) 내부 R&D, 3) Azure 순서로 배정하고 있습니다.

M365 코파일럿과 GitHub 코파일럿은 소모되는 컴퓨팅 캐파 대비 수익화 관점에서 Azure 클라우드보다 뒤처진 것으로 평가되었습니다. 이에 마이크로소프트는 코파일럿 수익화에 적극적으로 나서기 시작했습니다. M365 코파일럿의 유료 사용자 시트 수는 전년 대비 4배 증가한 2,000만 개를 기록했습니다. 7월 1일부터는 가격 인상이 예정되어 있으며, 사용자당 과금 구조로 ARPU 상승에 기여할 것으로 기대됩니다. 코파일럿 주간 참여도는 Outlook과 동일한 수준까지 상승했습니다.

한편, GitHub 코파일럿은 기존 사용자 과금 방식에서 사용량 기반 가격 책정으로 전환했습니다. 사용하는 토큰 수에 비례하여 과금하는 방식으로 전환하면서 매출 증대에 도움이 될 것으로 추론됩니다. 또한, 추론 비용 절감을 위해 코파일럿 워크로드에 특화된 자체 AI 모델 개발도 확대하고 있습니다. 음성-텍스트 변환 MAI-Transcribe와 이미지 MAI-Image-2가 대표적인 예입니다.

Azure 클라우드 성장률 가속화는 하반기에 나타날 전망

하반기에는 Azure 클라우드 성장률 가속화가 기대됩니다. 마이크로소프트는 금융리스를 포함한 CY 2026 CapEx 가이던스를 1,900억 달러로 제시했습니다. 부품 가격 인상 효과 250억 달러가 반영된 점을 감안하더라도 예상치를 상회하는 수치입니다. 참고로, 금융리스를 제외한 CY 2026 CapEx 컨센서스는 1,190억 달러 수준입니다.

직전 4개 분기 합산 금융리스는 300억 달러 수준이었으며, FY 1Q26 155억 달러 → FY 2Q26 76억 달러 → FY 3Q26 10억 달러로 감소하는 추세를 보였습니다. 이를 고려하면, 향후 데이터센터 건설 가속화를 통해 클라우드 성장률에 기여할 것으로 예상됩니다. 한편, 마이크로소프트도 알파벳, 아마존만큼은 아니지만 커스텀 AI 칩인 Maia, Cobalt 사용 확대를 통해 비용 절감을 추진하고 있습니다.

12개월 선행 P/E 22배로 밸류 부담은 낮으나 모멘텀은 하반기에 나타날 전망

현재 12개월 선행 P/E는 22배로 밸류 부담이 낮은 편입니다. 하지만 의미 있는 모멘텀을 확보하기까지는 다소 시간이 필요할 것으로 판단됩니다.
