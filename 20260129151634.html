FY 2Q26 Review: Azure 성장률이 고점을 형성했다는 아쉬움

FY 2Q26 매출액은 813억 달러(+16.8% YoY, +4.7% QoQ)로 컨센서스를 +1.2% 상회했습니다. 영업이익은 383억 달러(+21.0% YoY, +0.9% QoQ)로 +4.8% 상회했으며, EPS (GAAP 기준)는 5.16달러(+59.8% YoY, +38.7% QoQ)로 +34.7% 상회했습니다. EPS가 다른 지표 대비 컨센서스를 크게 상회한 것은 OpenAI 지분 가치 상승에 기인한 것으로 보입니다.

실적 발표 이후 시간 외 거래에서 -5% 하락했습니다. 이번 분기 실적은 예상을 상회하고 다음 분기 가이던스는 예상치에 부합한 수준이었으나, Azure 클라우드 성장률이 고점을 찍고 둔화될 것으로 전망되는 점이 부담으로 작용했습니다. 전년대비 분기별 Azure 매출 성장률은 FY 2Q25 31% → FY 3Q25 33% → FY 4Q25 39% → FY 1Q26 40% → FY 2Q26 39% → FY 3Q26E 37.5% (중간값 기준)를 기록했습니다. 참고로, 이번 분기 RPO는 6,250억 달러로 전년대비 110% 증가했으며, 이 중 12개월 이내에 매출 인식하는 cRPO는 전년대비 39% 증가했습니다. RPO 내 OpenAI 비중은 45% 수준입니다.

Azure 클라우드 성장률이 기대에 미치지 못하는 것은 마이크로소프트가 사업 부문의 균형 잡힌 발전을 추구한 결과입니다. 마이크로소프트는 데이터센터를 건설한 뒤 1) M365 코파일럿, GitHub 코파일럿과 같은 자사 앱, 2) 내부 R&D, 3) Azure 순서로 컴퓨팅 캐파를 할당하고 있습니다. CFO는 FY 1Q26, FY 2Q26에 가동된 GPU를 모두 Azure에 할당했을 경우 Azure 성장률은 40%를 상회했을 것이라고 코멘트했습니다.

한편, M365 코파일럿이 포함된 생산성 & 비즈니스 프로세스 부문의 매출 성장률은 가속화되고 있으나 그 속도가 가파르지는 않습니다. 전년대비 분기별 생산성 & 비즈니스 프로세스 매출 성장률은 FY 2Q25 13.7% → FY 3Q25 10.5% → FY 4Q25 15.7% → FY 1Q26 16.6% → FY 2Q26 16.0% → FY 3Q26E 14.9% (중간값 기준)입니다.

CapEx 증가에 따른 감가상각비 증대로 영업이익률은 하락할 전망

FY 3Q26 영업이익률은 전년대비 하락할 전망입니다. CapEx 증대로 감가상각비가 증가한 영향입니다. 마이크로소프트는 AI 투자에 따른 비용을 인력 효율화를 통한 영업비용 절감을 통해 이익률을 최대한 방어하려는 모습을 보이고 있습니다.

Azure 성장률이 고점을 형성하고 둔화될 가능성을 보이면서 멀티플 프리미엄을 받기는 어려운 구간으로 진입했습니다. 다만, Azure 및 생산성 & 비즈니스 프로세스 부문의 안정적인 성장세를 고려하면 추가 하락 시 밸류 매력이 부각될 것으로 판단됩니다. 마이크로소프트의 12MF P/E는 26배 수준으로 과거 받던 프리미엄 상당 부분을 반납했습니다.

컨퍼런스콜 정리

Q. 데이터센터

- 와트당 달러당 토큰(tokens per watt per dollar) 최적화를 추구하고 있습니다. 이는 실리콘 시스템과 소프트웨어를 사용하여 활용도를 높이고 TCO(총 소유 비용)를 낮추는 것으로 귀결됩니다. 일례로, Copilot을 구동하는 가장 많은 양의 워크로드 중 하나인 OpenAI 추론에서 50%의 처리량 향상을 이뤘습니다.

- Maia 200은 FP4 정밀도에서 10PFLOPS 이상의 성능을 제공하며, 최신 세대 하드웨어 대비 30% 이상 개선된 TCO를 제공합니다. Cobalt 200은 클라우드 네이티브 워크로드를 위해 자체 제작한 첫 프로세서보다 50% 이상 높은 성능을 제공합니다.

- 메모리 가격은 자본 지출에 영향을 미치겠지만, Microsoft Cloud 매출 총이익률에 미치는 영향은 자산이 6년에 걸쳐 감가상각됨에 따라 점진적으로 나타날 것입니다.

Q. Copilot

- Microsoft 365 Copilot은 일일 활성 사용자가 전년 대비 10배 증가하며 진정한 일상적 사용 문화가 정착되고 있습니다. 이번 분기는 Microsoft 365 Copilot 시트 추가가 전년 대비 160% 이상 증가하며 기록적인 분기를 기록했습니다. 분기별 시트 성장이 가속되었으며 현재 1,500만 개의 유료 Microsoft 365 Copilot 시트와 그 수배에 달하는 엔터프라이즈 채팅 사용자를 보유하고 있습니다.

- 유료 GitHub Copilot 전반에서 강력한 성장세를 보였습니다. 개인 개발자용 Copilot Pro Plus 구독은 전분기 대비 77% 증가했습니다. 전체 유료 Copilot 구독자는 전년대비 75% 증가한 470만 명입니다.

Q. RPO

- 상업적 잔여 이행 의무(RPO)는 6,250억 달러로 전년 대비 110% 증가했으며, 가중 평균 기간은 약 2.5년입니다. 이 중 약 25%가 향후 12개월 내에 매출로 인식될 예정이며, 이는 전년 대비 39% 증가한 수치입니다. 12개월 이후 인식될 잔여분은 156% 증가했습니다. 상업적 RPO 잔액의 약 45%는 OpenAI와 관련이 있습니다. 그 외 상당한 잔여 잔액은 28% 성장하며 포트폴리오 전반에 걸친 지속적이고 광범위한 고객 수요를 반영하고 있습니다.

- 구매하는 GPU의 상당 부분은 이미 그 내용연수의 대부분에 대해 계약이 완료된 상태입니다. 대형 고객들과 맺은 GPU 계약의 경우, GPU의 전체 내용연수 동안 판매됩니다. RPO가 짧게 나타나는 이유는 M365 관련 부분 때문이며, Azure 전용 RPO를 보면 기간이 조금 더 길게 나타납니다.

Q. Agent AI

- 이번 분기에 기업들이 기존의 거버넌스, ID, 보안 및 관리를 에이전트로 쉽게 확장할 수 있게 해주는 Agent 365를 출시했습니다. 이는 고객이 Microsoft 365와 Azure에서 이미 사용 중인 동일한 제어 기능을 당사 클라우드나 다른 클라우드에 배포된 에이전트에도 확장할 수 있음을 의미합니다. Adobe, Databricks, Genspark, Glean, NVIDIA, SAP, ServiceNow, Workday와 같은 파트너들이 이미 Agent 365를 통합했습니다.
