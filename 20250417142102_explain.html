대신증권 보고서 쉽게 읽기 (2025년 4월 17일)

핵심 요약

쉽게 말해, 이번 보고서는 우리나라 경제 상황과 관련된 이야기예요.

  • 기준금리 동결: 한국은행이 돈 빌리는 값(기준금리)을 그대로 뒀어요.
  • 추가 인하 가능성: 하지만 앞으로 돈 빌리는 값을 더 낮출 가능성이 있다고 봐요.
  • 5월 인하 전망: 다음 달(5월)에는 돈 빌리는 값을 낮출 거라고 예상해요.
  • 채권 투자 조언: 채권에 투자하려면 지금 바로 사기보다는 조금 기다렸다가 사는 게 좋을 것 같아요.

왜 기준금리를 그대로 뒀을까?

한국은행은 4월에 돈 빌리는 값, 즉 기준금리를 2.75%로 동결했어요. 쉽게 말해, 돈을 빌려주는 은행이 돈을 빌릴 때 내는 이자를 그대로 둔 거죠.

왜 그랬냐면, 요즘 돈의 가치가 왔다 갔다 하고, 사람들이 빚을 많이 지는 것 같아서 걱정했기 때문이에요. 하지만 경제가 잘 안 돌아갈 위험도 있어서, 돈 빌리는 값을 낮춰야 할 필요성도 크다고 생각하고 있어요.

*기준금리: 한국은행에서 정하는 돈 빌리는 값의 기준. 이 값이 낮아지면 은행들도 돈을 싸게 빌려줄 수 있어서, 사람들이 돈을 더 쉽게 빌릴 수 있게 돼요.
*동결: 변화 없이 그대로 유지하는 것.

앞으로 돈 빌리는 값을 더 낮출까?

많은 사람들이 한국은행이 돈 빌리는 값을 더 낮출 거라고 생각하고 있어요. 왜냐하면, 전에 돈 빌리는 값을 낮췄었고, 요즘 사람들이 빚을 많이 지는 것 같아서 조심해야 하지만, 경제가 어려워질 수도 있으니까요.

한국은행은 "경제가 안 좋아질 위험을 막기 위해 돈 빌리는 값을 낮추는 것을 계속 생각할 것"이라고 했어요. 그리고 앞으로 여러 가지 상황을 잘 살펴보고 돈 빌리는 값을 얼마나 더 낮출지, 언제 낮출지 결정할 거라고 했어요.

*통화정책방향: 한국은행이 앞으로 돈과 관련된 정책을 어떻게 할 건지 알려주는 자료.
*성장의 하방리스크: 경제가 예상보다 안 좋아질 위험.

5월에는 돈 빌리는 값을 낮출 것 같아요

이번에 돈 빌리는 값을 그대로 뒀지만, 경제가 안 좋아질 위험이 크기 때문에 한국은행은 앞으로 돈 빌리는 값을 더 낮출 가능성이 크다고 보고서는 말하고 있어요. 특히 다음 달인 5월에 돈 빌리는 값을 낮출 가능성이 크다고 예상하고 있어요.

하지만 돈 빌리는 값을 너무 많이 낮추지는 않을 것 같아요. 왜냐하면, 돈 빌리는 값을 너무 많이 낮추면 다른 문제가 생길 수도 있으니까요.

채권 투자, 어떻게 해야 할까요?

채권은 돈을 빌려주고 받는 증서 같은 거예요. 돈 빌리는 값을 낮추면 채권 가격이 올라갈 가능성이 커요.

하지만 이미 채권 가격이 많이 오른 상태라서, 지금 바로 채권을 사기보다는 조금 기다렸다가 가격이 내려갔을 때 사는 게 좋을 것 같아요.

*채권: 정부나 회사가 돈을 빌리고 나중에 돈을 갚겠다는 약속이 적힌 종이. 채권을 사면 나중에 이자를 받을 수 있어요.
